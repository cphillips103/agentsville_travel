create new environment in anaconda navigator

open powershell using admin rights

conda activate "environment name"

install ipykernel next:

conda install -c anaconda ipykernel

After installing this,

just type,

python -m ipykernel install --user --name="environment name"

also install jupyter:

conda install jupyter

Can set conda-forge parameter:
conda config --add channels conda-forge
conda config --set channel_priority strict

If conda has compatibility issues installing a package (example):
conda install numexpr python=3.13.5
